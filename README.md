## 🛠️ Công nghệ

Đơn giản thôi — chỉ 3 file:

| File | Vai trò |
|------|---------|
| `index.html` | Nội dung & cấu trúc |
| `styles.css` | Giao diện & dark mode |
| `script.js` | Tương tác & animation |

Không framework, không build step — mở file HTML là chạy. 🚀
